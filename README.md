# Spring Boot for Beginners

## How to run Docker
Run Docker
```
docker compose up -d
```
More commands
```
docker compose ps
```
```
docker logs postgres -f
```
```
 docker exec -it postgres bash
```
<small>CTRL + D to exit root container</small><br>
Manage Database
```
psql -U _USERNAME_
```
<small>CTRL + D to exit manager database</small><br>
